Since I'm sure we're all going to be playing some blackjack on our upcoming trip, I thought I'd post the basic strategy for the game.

Blackjack is a fairly straightforward game, so computers are able to determine the best strategy for playing. The chart below lays out this computer-generated strategy.

To read the chart, simply find your hand in the left-hand column, then find the dealer's up card on the top row, and refer to the spot in the chart where the two meet. For example, if you have 11, and the dealer has a 9, you should double down. If you have A-7, and the dealer has an 8, you should stay. If you have 3-3 and the dealer has a 4, you should split. And so on.

Try memorizing this chart; it won't take you that long. And if you'd like to test yourself on basic strategy, go to this website: http://blackjack-basics.com/bjd/bsdrill.php3
